oneWayBranch

inline fun <From, To> IRoBox<From>.oneWayBranch(crossinline reduce: From.() -> To): IRoBox<To>

Returns a box that has a one-way binding to this one. I.e. if this one changes another box gets changes as well. However if another box changes this one remains unchanged.

Parameters

reduce - a function to transform the payload of this box into the payload of the other box
