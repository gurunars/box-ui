Returns a box that has a two-way binding to this one. I.e. if this one changes another box gets changed and vice versa.

Parameters

reduce - a function to transform the payload of this box into the payload of the other box

patchSource - a function to transform the value of this box based on the value of the other box

inline fun <Type> IBox<Optional<Type>>.branch(initial: Type): IBox<Type>

Returns a special binding that propagates updates only for non null (Optional.Some) values.

Parameters

initial - the value that is set before any of the events are propagated
