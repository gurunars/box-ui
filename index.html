/ index

A collection of modules aiming to simplify creation of very specific simple CRUD apps for Android.

In order to fetch the modules as dependencies add the following to your top level build.gradle: 

allprojects {
    repositories {
        ...
        maven { url "http://dl.bintray.com/berezin/maven" }
        ...
    }
}

android-utils

A set of helpers for working with Android. 

implementation ('com.gurunars.android_utils:android-utils:0.196.0@aar') {
    transitive = true
}

box

A library to facilitate data binding via observable data containers that can be entangled together, manipulated and monitored for changes. 

implementation 'com.gurunars.box:box:0.196.0'

box-ui

Android specific databinding functionality. 

implementation ('com.gurunars.box.ui:box-ui:0.196.0@aar') {
    transitive = true
}

crud-item-list

One more layer of abstraction above the item-list component. It contains the logic and UI controls to manipulate individual items in the list. I.e. the UI allows to select, move, edit, remove and create the items out of the box. 

implementation ('com.gurunars.crud_item_list:crud-item-list:0.196.0@aar') {
    transitive = true
}

floatmenu

A component for creating FAB based menus to manipulate the app. In contrast with the toolbar it aims to simplify the way the app can be managed with left or right hand thumb on a large phone screen. 

implementation ('com.gurunars.floatmenu:floatmenu:0.196.0@aar') {
    transitive = true
}

item-list

A component based on RecyclerView with a simplified interface capable of detecting payload changes in the collection items and triggering proper animations based on those changes. Apart from the animations the component handles custom empty pages and can optionally handle item selection. 

implementation ('com.gurunars.item_list:item-list:0.196.0@aar') {
    transitive = true
}

storybook

A library to help developing custom box-ui components via a registry based storybook activity. 

implementation ('com.gurunars.storybook:storybook:0.196.0@aar') {
    transitive = true
}

storybook-annotations

A StorybookComponent annotation to be used together with storybook and storybook-registry. 

implementation 'com.gurunars.storybook_annotations:storybook-annotations:0.196.0

storybook-registry

A storybook-registry annotation processor to be used together with storybook and storybook-annotation. 

kapt 'com.gurunars.storybook_registry:storybook-registry:0.196.0'
