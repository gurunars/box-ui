floatMenu

fun Context.floatMenu(contentPane: IRoBox<ContentPane>, menuPane: IRoBox<MenuPane>, animationDuration: Int = 400, isOpen: IBox<Boolean> = Box(false)): StatefulView

Floating menu available via a FAB

Parameters

contentPane - View shown in the background layer of the widget. Semantically it represents the data manipulated by the menu.

menuPane - View shown in the foreground layer of the widget when the menu is open. Is supposed to contain menu's controls.

animationDuration - Time it takes to perform all the animated UI transitions.

isOpen - flag indicating visibility of the menu pane on the screen
