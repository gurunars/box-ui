selectableItemListView

fun <ItemType : Item> Context.selectableItemListView(items: IRoBox<List<ItemType>>, selectedItems: IBox<Set<ItemType>> = Box(setOf()), itemViewBinders: Map<Enum<*>, ItemViewBinder<SelectableItem<ItemType>>> = mapOf(), emptyViewBinder: EmptyViewBinder = this::defaultEmptyViewBinder, explicitSelectionMode: IRoBox<Boolean> = false.box): View

Item list that has selection enabled.

Items can be selected initially via long clicking and further on by consequentially clicking them.

Parameters

ItemType - type of the item to be shown in the list

items - A collection of items shown in the list

selectedItems - A collection of items selected at the moment

itemViewBinders - a type based mapping between item type and item renderer

emptyViewBinder - a function returning a view to be shown when the list is empty

explicitSelectionMode - when true selection mode is initiated via normal click instead of a long one
